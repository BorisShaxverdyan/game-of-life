# Game of life

## How to start project

Go to any folder you want. The folder must be empty.

Copy the project to that folder:

```
$ git clone https://github.com/BorisShaxverdyan/game-of-life.git .
```

Install all npm dependencies.

```
$ npm i
```

### To run project

Run this command:

```
$ npm run build
```

And then run server using this command:

```
$ npm run server
```

Then go to http://localhost:3000/ page

### To start development

Run this command in one bash tab:

```
$ npm run watch
```

And run this command in another bash tab:

```
$ npm run server
```

Then go to http://localhost:3000/ page.
